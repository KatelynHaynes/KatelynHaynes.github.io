Bolted Revenue: Elysian Fly Company

Course: MIS 3353 Database Management

Team Members: Andrea Barcenas, Cole Fielding, Katelyn Haynes, Matthew Nguyen, Joshua Sowell

This project involved designing and implementing a database solution for Elysian Fly Company, focusing on inventory and vendor management, sales processes, and data optimization. Key achievements included:

  • Developing an Entity-Relationship Diagram (ERD) to organize and map data relationships.
  • Implementing normalization to reduce redundancy and improve database efficiency.
  • Creating a data dictionary for better data governance and consistency.
  • Integrating business cycles like revenue and expenditure into the database design.
